# EDR-006 — Não reduzir fonte para resolver overflow
- **context**: EDS-007/EDS-012/EDS-041 — compressão para manter "1 de 1" gerou ilegibilidade.
- **problem**: overflow tratado por encolhimento tipográfico.
- **decision**: overflow **MUST** ser resolvido por REPRIORITIZE → SUBSTITUTE → PAGINATE → sinalizar; **MUST NOT** reduzir fonte abaixo do mínimo do perfil nem usar SCALE para caber (TR-002).
- **positive_consequences**: legibilidade preservada; nada crítico cortado.
- **negative_consequences**: mais páginas/superfícies.
- **requirement_refs**: [FR-007, PRINT-002]
- **test_refs**: [TEST-TR-02, TEST-PRINT-01]
