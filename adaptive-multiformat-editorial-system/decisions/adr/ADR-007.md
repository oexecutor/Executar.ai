# ADR-007 — Saída estruturada paralela
- **context**: EDS-031 — visual não é contrato para agentes.
- **problem**: como garantir leitura por agentes?
- **decision**: toda saída visual é acompanhada de **dados estruturados paralelos** (JSON/CSV) referenciados no output.
- **positive_consequences**: sem OCR; automação; auditoria.
- **negative_consequences**: dobra de artefatos a manter em sincronia.
- **requirement_refs**: [FR-005, AI-001]
- **test_refs**: [TEST-AI-01]
