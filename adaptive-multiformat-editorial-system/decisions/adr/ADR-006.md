# ADR-006 — P-1 vs P-2
- **context**: SRC-01/SRC-05: "salvar como PDF" do navegador não é fluxo gráfico profissional.
- **problem**: como não prometer gráfica onde só há export de navegador?
- **decision**: separar formalmente **P-1** (navegador, window.print, @media print, @page, sem garantia PDF/X) de **P-2** (PDF/X, CMYK, ICC, sangria, fontes incorporadas, 300dpi, preflight, prova, imposição).
- **positive_consequences**: expectativa correta; preflight explícito.
- **negative_consequences**: P-2 exige pipeline externo não incluído.
