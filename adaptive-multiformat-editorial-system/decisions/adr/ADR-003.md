# ADR-003 — Perfis de formato como objetos versionados
- **context**: SRC-01 exige que formato seja objeto de configuração, não classe CSS.
- **problem**: como representar A4, livro, panfleto, widget, BMC, cartão, mobile, desktop, slide e imagens?
- **decision**: cada formato é um **objeto de perfil versionado** em `format-profiles.yaml` com dimensões, grid, orçamento, comportamentos preferidos/proibidos e print_class.
- **positive_consequences**: adaptação declarativa; auditável; todos os formatos P0 tratados igualmente.
- **negative_consequences**: perfis sem amostra real ficam PROVISIONAL (LAC-009).
