# ADR-005 — Fonte modular; bundle único apenas para distribuição
- **context**: SRC-05 §25 proíbe HTML monolítico como fonte de desenvolvimento.
- **problem**: como estruturar código de saída?
- **decision**: a **fonte é modular** (conteúdo, tokens, componentes, perfis, estilos). Um HTML único MAY existir **apenas** como artefato final de distribuição.
- **positive_consequences**: manutenível; sem JSX gigante; build controlado.
- **negative_consequences**: exige passo de empacotamento (G7).
